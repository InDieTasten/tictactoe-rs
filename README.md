Tic Tac Toe aka knots and crosses in Rust
=========================================

This is an educational project for me to learn Rust with.

Setup
-----

```bash
git clone https://github.com/InDieTasten/tictactoe.rs
cd tictactoe.rs
cargo build
```

Usage
-----

### Play local player vs player
```bash
cargo run
```

### Play local player vs AI
```bash
cargo run -- --ai=1
```
